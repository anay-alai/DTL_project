# JEE(Main) Website Clone

The **JEE(Main) website lookalike** replicates the simple but powerful frontend almost all engineering aspirants are familiar with. 

## Features

- The homepage includes clickable logos of **NTA** and **MOE**.
- A **Result Page** is implemented which:
  - Verifies the Application Number and Password using JavaScript.
  - Grants access to the result if:
    - **Application Number:** `12345`
    - **Password:** `secret`
- The **About Us** page includes:
  - A slideshow featuring various **IIT campuses**.
  - A table showcasing **NIRF rankings** of select IITs/NITs along with their respective states.
- The **FAQ Page**:
  - Contains a dropdown menu where answers are initially hidden and revealed upon selection.

## Technologies Used

- **HTML**
- **CSS**
- **JavaScript**
